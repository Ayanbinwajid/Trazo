Developed a ride-booking platform similar to Uber and Rapido using the MERN stack (MongoDB, Express.js, React.js,
Node.js). Integrated Google Maps API for real-time location tracking and route visualization. Implemented user
authentication, ride management, and dynamic fare calculation features. Designed a responsive and interactive UI in React.js
to ensure a seamless user experience. Deployed and tested the application to maintain performance, reliability, and
scalability.
